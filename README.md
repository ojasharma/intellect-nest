## 📦 Getting Started

### Prerequisites

* Node.js >= 18.x
* npm or yarn

### Installation

```bash
git clone https://github.com/your-username/intellect-nest-landing.git
cd intellect-nest-landing
npm install
# or
yarn install
```

### Running the Dev Server

```bash
npm run dev
# or
yarn dev
```

Visit `http://localhost:3000` to view the site.
